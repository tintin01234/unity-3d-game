# Juego 3D en Unity

Videojuego 3D simple desarrollado con Unity y C# como proyecto de aprendizaje.

## Características del Juego

- Sistema de puntuación
- Detección de colisiones
- Enemigos con IA básica
- Sistema de tienda in-game
- Manejo de prefabs y assets

## Tecnologías

- **Motor:** Unity 2022.x
- **Lenguaje:** C#
- **Plataforma:** PC (Windows)

## Estructura del Proyecto
```
Assets/
├── Scripts/        # Código C#
├── Prefabs/        # Objetos reutilizables
└── Scenes/         # Escenas del juego
```

## Controles

- **WASD:** Movimiento
- **Espacio:** Saltar
- **Mouse:** Cámara
- **E:** Interactuar

## Cómo Ejecutar

1. Abre el proyecto en Unity Hub
2. Abre la escena principal: `Assets/Scenes/MainScene.unity`
3. Presiona el botón ▶️ Play
